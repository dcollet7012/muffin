# MUFFIN : Manage Unified File Flow In Network


# Features

- Embedded Server
  - FTP/FTPS Server
  - SFTP/SSH Server
  - AS2/3/4 Server
  - IMAP Server
  - SMTP Server

- Resources
  - File System Folder
  - AS3 Bucket
  - Mailbox
  - Database
  - Rest Server
  - SOAP Server
  - SSH Server
  - Webdav
  
- Translation
    - XML Reader/Writer
    - JSON Reader/Writer
    - CSV Reader/Writer
    - Excell Reader/Writer
    - Compression
